Politique de Confidentialité

Dernière mise à jour : 22 septembre 2025

Bienvenue

PurrrrrFocus (ci-après dénommé "nous" ou "cette application") prend la protection de votre vie privée très au sérieux. Cette politique de confidentialité précise comment nous protégeons vos informations personnelles. En utilisant cette application, vous acceptez les pratiques décrites dans cette politique de confidentialité.

Collecte d'Information

Cette application est conçue avec un stockage complet local et ne collecte aucune information personnelle identifiable :

  1. Données Locales : Les tâches que vous créez, les enregistrements de concentration, les préférences de paramètres, etc., sont stockés uniquement localement sur votre appareil.
  2. Aucune Collecte de Données : L'application ne collecte aucune statistique d'utilisation ou données d'analyse.
  3. Aucune Transmission Réseau : Toutes les données restent sur votre appareil et ne sont jamais téléversées sur aucun serveur ou tiers.
  4. Achat Intégrés : L'application propose une fonctionnalité d'achat intégré de la version Plus. Le traitement des paiements est effectué par Apple App Store, et nous ne collectons ni ne stockons vos informations de paiement.

Utilisation des Données

Puisque l'application utilise un stockage complet local, nous ne collectons aucune information. La fonctionnalité de l'application est entièrement basée sur les données locales de votre appareil :

  • Fournir les fonctionnalités de l'application : Sur la base des données locales de votre appareil
  • Fournir l'assistance client : Par le biais de la fonction de retour à l'intérieur de l'application
  • Assurer la sécurité et la stabilité de l'application : Par les mécanismes de sécurité du système iOS
  • Respecter les obligations légales : Suivre les lois et réglementations pertinentes

Note : Tous les traitements de données se font sur votre appareil et ne sont jamais téléversés sur aucun serveur.

Stockage des Données

  • Base de Données Locale : Utilise le framework iOS CoreData pour stocker vos enregistrements de concentration, vos données de tâches, etc.
  • Paramètres Système : Utilise iOS UserDefaults pour stocker les paramètres et préférences de l'application
  • Local de l'appareil : Toutes les données sont stockées uniquement sur votre appareil
  • Aucune Synchronisation Cloud : Les données ne sont pas synchronisées avec iCloud ou d'autres services cloud
  • Export des Données : La version Plus permet d'exporter les données au format CSV/Excel, les fichiers sont stockés dans le répertoire temporaire local de l'appareil
  • Suppression des Données : Vous pouvez supprimer les données locales à tout moment via les paramètres de l'application

Partage des Données

Nous ne vendons, n'échangeons ni ne transmettons vos informations personnelles à des tiers. Puisque l'application utilise un stockage complet local, nous ne partagerons des informations que dans les cas suivants :

  • Avec votre consentement explicite
  • Pour respecter les exigences légales ou les ordonnances judiciaires
  • Pour protéger nos droits, notre propriété ou notre sécurité

Fonctionnalités de Partage Initiées par l'Utilisateur :

  • Fonction d'Export des Données : La version Plus permet d'exporter vos données de concentration au format CSV/Excel. Les fichiers exportés sont stockés dans le répertoire temporaire local de l'appareil, et vous pouvez choisir de partager les données avec d'autres applications via la fonctionnalité de partage système
  • Fonction de Partage de l'Application : Vous pouvez partager des recommandations d'application à d'autres utilisateurs. Le contenu partagé inclut les liens App Store et ne contient pas de données personnelles
  • Partage de Cartes de Succès : Vous pouvez partager l'information de débloquement de succès sur les plateformes sociales. Le contenu partagé inclut les noms et descriptions des succès, mais ne contient pas de statistiques personnelles
  • Enregistrement d'Images de Succès : Vous pouvez enregistrer les cartes de succès dans l'album photo de l'appareil. Les images contiennent les noms et descriptions des succès, mais ne contiennent pas de statistiques personnelles

Note : Outre les comportements de partage initiés par l'utilisateur ci-dessus, nous ne partageons pas automatiquement vos données. L'utilisation des données après le partage est contrôlée par vous et les destinataires, et nous n'y participons pas.

Sécurité des Données

Nous prenons des mesures techniques appropriées pour protéger vos informations personnelles :

  • Stockage Local : Toutes les données sont stockées uniquement sur votre appareil et ne sont jamais téléversées sur aucun serveur
  • Protection Système : Les données sont protégées par les mécanismes de sécurité du système iOS
  • Isolation des Données : Les données de l'application sont complètement isolées des données d'autres applications
  • Aucune Transmission Réseau : Toutes les données ne sont jamais transmises via le réseau

Vos Droits

Puisque cette application utilise un stockage complet local, vous disposez d'un contrôle total sur vos données :

  • Accès aux Données : Toutes les données sont stockées sur votre appareil et vous pouvez les consulter à tout moment
  • Modification des Données : Vous pouvez modifier ou supprimer n'importe quelle donnée à tout moment
  • Export des Données : La version Plus permet d'exporter les données de statistiques de concentration au format CSV/Excel
  • Réinitialisation des Données : Vous pouvez réinitialiser toutes les données de l'application à tout moment
  • Suppression des Données : La suppression de l'application supprime complètement toutes les données

Cette application ne requiert pas de fonctionnalité de suppression de compte car il n'y a pas de comptes côté serveur.

Vie Privée des Enfants

Cette application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons délibérément aucune information personnelle d'enfants de moins de 13 ans. Si nous découvrons que nous avons collecté des informations d'un enfant de moins de 13 ans, nous la supprimerons immédiatement.

Changements de la Politique

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. Les changements significatifs seront communiqués via des notifications dans l'application ou des journaux de mise à jour. L'utilisation continue de l'application indique votre acceptance de la politique mise à jour.

Contactez-nous

Si vous avez des questions concernant cette politique de confidentialité, veuillez nous contacter via :

  • Email : support@louisbuilds.me
  • Retour à l'intérieur de l'application : Par le biais de la fonction de retour dans la page des paramètres de l'application

Nous vous répondrons à votre demande dans les plus brefs délais.
